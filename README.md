# C++ Kata
Just for fun...

# Let's start...

## Requirements
* compiler with c++14 support
* [cmake](https://cmake.org)

## Configure

    git clone [https://github.com/w07c1ech/kata.git](https://github.com/w07c1ech/kata.git)
    cd kata
    mkdir build && cd build
    cmake . ..

## Build and run

    cd build
    make
    ./kata # or ctest
